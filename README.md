# ShotSage Website

Marketing / brand storytelling site for [shotsage.ai](https://shotsage.ai).

## Stack
- Single-file HTML — no build step, no dependencies
- Fonts loaded from Google Fonts CDN (Playfair Display, DM Sans, JetBrains Mono)
- All animations via pure CSS

## Deploy
Hosted on Vercel. Push to `main` to auto-deploy.

## Local Preview
Just open `index.html` in a browser.
